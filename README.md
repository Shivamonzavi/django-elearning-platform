# Django E-learning Platform

## Project Overview

A web-based e-learning platform developed with Django that allows instructors to create courses and students to enroll and access course content. The project demonstrates full-stack web development concepts including authentication, database management, and course administration.

---

## Features

- User authentication and authorization
- Course creation and management
- Student enrollment
- Course content organization
- Admin interface for managing data

---

## Technologies

- Python
- Django
- SQLite
- HTML
- CSS
- Bootstrap

---

## Project Structure

- Django application
- Database models
- Authentication system
- Course management
- Templates and static files

---

## Purpose

This project was developed as part of the M.S. Information Systems program to demonstrate practical experience with web application development using the Django framework.
